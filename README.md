# Tonight, Not Wasted

Initial crowdfunding landing-page draft for the course project.

## Current scope

This version focuses only on:

- the webpage body for the crowdfunding proposal
- the eight rhetorical moves required by the assignment
- a lightweight repository structure for future iteration

## Files

- `index.html`: landing-page draft
- `styles.css`: visual system and responsive layout
- `script.js`: light interaction for navigation and section reveal
- `assets/`: reserved for future images and video
- `docs/`: reserved for later process evidence and reflections

## Positioning

The project is not framed as a student-run food bank. It is presented as a small-scale, AI-assisted pilot toolkit that helps campus or community partners coordinate the transfer of still-edible short-dated food in a safer and more transparent way.
