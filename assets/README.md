# Visual Assets

This folder stores the visual materials used by the crowdfunding webpage.

## Files

- `readme-banner.svg`: README campaign banner.
- `videos/convenience-store-night.mp4`: hero campaign reel showing closing-time shelves.
- `videos/closing-leftovers.mp4`: campaign reel showing unsold food near closing time.
- `videos/campus-waste-seen.mp4`: campaign reel showing campus food-waste context.

The video files are kept under `assets/videos/` to follow the project requirement that AI-generated or AI-assisted visual
materials should be stored in the assets folder.
